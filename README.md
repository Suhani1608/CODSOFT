# CODSOFT
# CodSoft AI Internship Portfolio

This repository contains the projects and tasks completed during my Artificial Intelligence internship at **CodSoft**. The portfolio demonstrates practical implementations of Natural Language Processing (NLP), Search Algorithms, and Machine Learning.

## 📂 Repository Structure

### 1. `task1_chatbot/`
A rule-based conversational agent built using Python. It utilizes the `re` module for regular expressions and pattern matching to identify user intent and provide contextually appropriate responses without relying on heavy external NLP libraries.

### 2. `task2_tictactoe/`
An AI agent programmed to play the classic game of Tic-Tac-Toe. It showcases foundational AI decision-making logic and algorithmic game theory. 

### 3. `task4_recommendation_system/`
A Content-Based Movie Recommendation System. It processes the TMDB 5000 Movies Dataset using `pandas`, applies `TfidfVectorizer` to extract mathematical features from movie plot overviews, and computes `cosine_similarity` via `scikit-learn` to suggest highly relevant titles to the user based on their preferences.
